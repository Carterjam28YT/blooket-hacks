How to install

No extensions needed. Just a bookmark. Works in Chrome, Firefox, and Edge.

01
Grab the bookmarklet file
Download bookmarklet.js from the GitHub repo. Open it in any text editor and select all + copy the whole thing.
02
Make a new bookmark
Press Ctrl+Shift+B to show your bookmarks bar, right-click it and hit "Add page" or "Add bookmark". Call it whatever you want.
03
Paste the code as the URL
Edit the bookmark, clear the URL field, then paste the whole file contents in. It starts with javascript:(. Save it.
04
Click it on Blooket
Go to blooket.com or join a game, then click the bookmark. The menu pops up — drag it with the blue handle or top bar. Ctrl+E hides it, Ctrl+X closes it completely.
